# GenshinImpactSimulatedWish
 用纯 HTML5 搭建的原神祈愿组件演示站点，拥有一致的 UI 体验。
## 实现
### 动画
- Animate api ： 57.14%
- CSS变量 + Js ：41.27%
- 媒体 ：1.59%
- 纯CSS (除修饰器) ：0%

### 随机概率
 - 5星（除保底）：1%
 - 4星（除保底）：9%
 - 10 连则必出 4 星以上物品
 - 每 80 次必出一5星
 - 保底机制不影响概率
 - 抽卡次数不影响概率
 - 命定值为 1 时，下一次 5 星必为定轨物品
